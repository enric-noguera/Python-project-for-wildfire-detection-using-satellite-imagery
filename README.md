# Wildfire Detection using Landsat-8 Imagery

Python project for assessing wildfire impact using Landsat-8 satellite images from the 2019 Torre de l'Espanyol wildfire (Catalonia, Spain).

## Features

- Calculation of NDVI, NBR, SAVI, EVI and GCI vegetation indices.
- Burn severity assessment using dNBR.
- Estimation of burned area.
- Random Forest model for burn severity prediction.
- Visualization of satellite imagery and results.

## Technologies

- Python
- NumPy
- Matplotlib
- Scikit-Learn

## Results

The project generates:

- NDVI and NBR maps
- Burn severity classification maps
- Burned area estimation
- Random Forest predictions

## Data

The original Landsat-8 images are not included due to their size. They can be downloaded from the USGS EarthExplorer platform.
